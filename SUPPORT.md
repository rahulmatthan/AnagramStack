# Anagram Stack Support

Need help with Anagram Stack? Use one of the options below.

## Contact

- Support / bug reports: https://github.com/rahulmatthan/AnagramStack/issues

## What to include in a report

To help diagnose issues quickly, include:

- device model (for example, iPhone 14)
- iOS version
- app version and build number
- clear reproduction steps
- screenshots or screen recording (if possible)

## Common topics

- chain download issues
- saved progress issues
- hint behavior
- UI or accessibility feedback
